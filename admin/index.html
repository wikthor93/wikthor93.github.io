Wpisy blogowe

Edytujesz teksty interfejsu (menu, przyciski, opisy). Zmiany zapisują się lokalnie. Kliknij Eksport aby pobrać plik.

Konfiguracja formularza kontaktowego

  1. Idź na formspree.io i utwórz darmowe konto
  2. Kliknij "New Form", podaj swój adres email
  3. Skopiuj ID formularza (np. xpzgkwqv)
  4. Wklej ID poniżej i kliknij Zapisz
  5. Plik index.html zostanie zaktualizowany

Eksport treści

Pobierz plik content.js z wszystkimi wpisami, materiałami i tłumaczeniami. Zastąp nim plik na serwerze.

Import treści

Wgraj wcześniej wyeksportowany plik content.js aby przywrócić treści.
